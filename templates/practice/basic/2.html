Practice/Basic-2 - "Hello, World!" を実行してみよう

概要

最初は難しいことを考えずに、プログラミングの基本「Hello, World!」を実行してみましょう。

サンプルコード

1. Hello, World!

まずは、以下のソースコードをコピー＆ペーストして実行してみましょう。

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

2. 実行結果

IDEで実行してみましょう。すると以下のような実行結果が得られるはずです。

Hello, World!
